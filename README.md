# ncurses-pony
An ncurses library for pony

## Usage
Using [pony-stable](https://github.com/jemc/pony-stable) for dependency management, add the following to your `bundle.json` file:

```
{
  "deps": [
    { "type": "github",
      "repo": "jtfmumm/ncurses-pony",
      "subdir": "src/",
      "tag": "0.0.2"
    }
  ]
}
```
